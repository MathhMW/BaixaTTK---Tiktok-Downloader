# BaixaTTK - Downloader de TikTok

## Descrição
Este aplicativo permite baixar vídeos e áudios do TikTok de forma simples e rápida, com interface gráfica intuitiva.

---

## Como usar
1. Abra o executável BaixaTTK.exe  
2. Cole o link do TikTok  
3. Escolha o formato desejado (OGG ou MP4)  
4. Aguarde o download e pronto!

---

## Funcionalidades
- Download de vídeos (MP4)
- Download de áudio (OGG)
- Barra de progresso em tempo real
- Interface simples e rápida
- Bloqueio de múltiplos cliques durante o download

---

## Limitações
- Suporte apenas para vídeos de até 1 hora
- Dependente da disponibilidade do TikTok

---

## Tecnologias e ferramentas utilizadas
- Python  
- yt-dlp (download de mídia)  
  https://github.com/yt-dlp/yt-dlp  

- FFmpeg (processamento de áudio e vídeo)  
  https://ffmpeg.org/  

- Tkinter (interface gráfica)  
- PyInstaller (geração do executável)  

- UPX (compressão do executável)  
  https://upx.github.io/
